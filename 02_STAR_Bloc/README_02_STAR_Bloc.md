# 02_STAR_Bloc — Bloc STAR (multi-winner, majoritarian)

**Bloc STAR** fills multiple seats by running single-winner STAR repeatedly:
elect a winner, remove them, re-run on the same ballots. The same majority can
win every seat — that's the point (and the caveat): Bloc is **majoritarian**,
not proportional. For proportional seats, see
[`../03_STAR_PR/`](../03_STAR_PR/).

Cases live in [`_main/`](_main/). Concept docs:
[`../00_start_here/proportional_representation/`](../00_start_here/proportional_representation/)
(which contrasts Bloc against the proportional methods).

(start with [What's so good about STAR](../00_start_here/STAR_Voting/whats_so_good_about_STAR_Voting.md)).
