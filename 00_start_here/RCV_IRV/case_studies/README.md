# RCV-IRV case studies — real elections, real properties

The properties this library demonstrates with small YAML elections — center squeeze, spoilers, non-monotonicity, exhausted ballots — have all happened in **real public elections**. These pages document the record.

## The pages

- [Alaska 2022](RCV_IRV_alaska_2022.md) — a real RCV-IRV center squeeze, spoiler, *and* monotonicity failure in one special election.
- [The Alternative Vote in Australia](RCV_IRV_australia.md) — what a century of actual use does and doesn't prove.
- [RCV-IRV in North Carolina](RCV_IRV_north_carolina.md) — the short-lived 2006–2013 experiment.
- [RCV-IRV — origins and spread](RCV_IRV_history.md) — where the method came from and how it spread.
