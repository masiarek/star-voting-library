# Start Here

Welcome to the **STAR Voting education library**. This repo teaches STAR in depth
and — for honest comparison — the other single-mark and ranked methods, with a
real tabulation engine, runnable example elections, and a test suite behind every
result.

## Pick your path

- **New to voting reform? Start with STAR** →
  [**STAR — Lesson 0**](STAR_Voting/STAR_start_here.md) (the "why" before the
  "how"), then the [STAR concept pages](STAR_Voting/).
- **Persuasion / debate prep** → [Why STAR Voting](Why_STAR_Voting.md).

## The methods (one page each)

| Method | Ballot | Page |
|---|---|---|
| **STAR** | score 0–5, then automatic runoff | [STAR_Voting/](STAR_Voting/) |
| **Approval** | approve any number (0/1) | [Approval_Voting/](Approval_Voting/approval_voting.md) |
