# Edelman's "Myth of the Condorcet Winner," counted — BV2173, live

The tabulatable evidence behind [The strongest case against the Condorcet winner](../../00_start_here/topics/condorcet/edelman_condorcet_myth.md). Paul H. Edelman (Vanderbilt, mathematics **and** law; 22 Supreme Court Economic Review 207 (2015)) argues the Condorcet winner shouldn't be privileged — via a "cancellation" profile that goes back to **Condorcet himself** (aimed, then, at Borda) and via the no-show/join-consistency theorems. Unlike the [FairVote article](../../00_start_here/topics/condorcet/fairvote_condorcet_claim_check.md), this one's math is right — which makes it the perfect steelman: on its own centerpiece profile, every **majoritarian** count (Ranked Robin, RCV-IRV, STAR's runoff) elects Ada and every **positional** count (Borda, score sum, Choose-One) elects Ben. Who's "right" is a 240-year-old argument, live.
