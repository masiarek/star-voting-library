# symmetric_centrist_bv2170 — one electorate, four tabulations (a symmetric center squeeze)

The textbook **two-poles-plus-a-centrist** profile (47/47/3/3), reproduced on BetterVoting (election `pp2q4q`, Test ID **BV2170**). 100 voters rank three candidates; the centrist **Casey** is the Condorcet winner (beats each pole 53–47) but has only 6 first choices. STAR and Ranked Robin elect Casey; RCV-IRV and Choose-One squeeze the centrist out and, because the electorate is perfectly symmetric between the poles, **deadlock the two poles in an exact tie** (resolved at random on BV).
