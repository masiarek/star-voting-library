# symmetric_centrist_all_methods — the Condorcet centrist, every BV method, two sizes

One symptom, the full BetterVoting method sweep, at two electorate sizes. Avery (left pole), Blake (right pole), Casey (centrist). Casey is the **Condorcet winner** but has the **fewest first choices**, so the three first-choice methods (RCV-IRV, STV, Choose-One) squeeze Casey out and — because the poles are a perfect mirror image — deadlock in an exact tie, while the four whole-ballot methods (STAR, STAR-PR, Approval, Ranked Robin) elect Casey.
