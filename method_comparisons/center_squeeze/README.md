# Center squeeze — RCV-IRV eliminates the consensus, STAR elects it

The same 1-D electorate, counted two ways. A broadly-liked **moderate** is almost
everyone's first or second choice, but few voters' *top* choice — so RCV-IRV
eliminates them early (the "center squeeze"), while STAR's automatic runoff elects
them. The canonical single-picture argument for STAR over IRV.

Read the reader-friendly **pages** (`center_squeeze_pages/`); the `.yaml` beside
each is the tabulatable source.
