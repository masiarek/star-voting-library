# sntv_village_council — SNTV, standalone (a concentrated minority wins a seat)

A clean, self-contained SNTV showcase: 9 residents, a 2-seat council. The Downtown majority (5) splits between Nora (3) and Omar (2); the Riverside minority (4) concentrates on Priya (4) — so **Priya and Nora win**, and the 44% minority earns a seat it would lose under block voting.
